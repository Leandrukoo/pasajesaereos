📄 utils.js — Funciones utilitarias compartidas
Este archivo contiene funciones de uso general disponibles en todas las páginas del sitio. Se carga antes que cualquier otro script de página para que sus funciones estén disponibles. Incluye: manejo de localStorage, validaciones de formularios, manipulación del DOM y notificaciones.
🔵 guardarDatos(clave, datos) ▾

Guarda cualquier dato en localStorage convirtiéndolo a texto JSON.

#CódigoExplicación
1 function guardarDatos(clave, datos) { DEF Declara la función. Recibe dos parámetros: clave (nombre identificador del dato) y datos (el valor a guardar, puede ser cualquier tipo).
2 try { Inicia un bloque try/catch. Si ocurre un error dentro (ej.: localStorage lleno), el programa no se rompe — lo captura el bloque catch.
3 localStorage.setItem(clave, JSON.stringify(datos)); LS JSON.stringify(datos) convierte el dato (objeto, número, array…) a un string de texto, porque localStorage solo puede guardar strings. Luego localStorage.setItem lo almacena bajo el nombre clave.
4 return true; RET Si todo salió bien, devuelve true para indicar éxito.
5 } catch (error) { ERR Si setItem lanzó un error, la ejecución salta aquí. error contiene los detalles del problema.
6 console.error(`✗ Error al guardar...`, error); Muestra el error en la consola del navegador para que el desarrollador pueda debuggear.
7 return false; RET Devuelve false para indicar que el guardado falló.
8 }
}		 Cierra el bloque catch y la función.
